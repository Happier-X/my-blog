可以通过 GitHub 提供的搜索功能，快速查找自己在 GitHub 发布过的 Issue。

1. 打开 [GitHub](https://github.com/)。
2. 在搜索框中输入 `is:issue involves:my-username(此处是你的 GitHub 用户名)`，查询即可。

例如，我想查找我发布的 Issue，可以输入如下查询语句。

```txt
is:issue involves:HappierPuppy
```
