## 使用 Vite 创建

命令：

```shell
npm create vue@latest
```

## 项目结构

```
├── node_modules
├── public
├── src
│   ├── assets
│   ├── components
│   ├── App.vue
│   └──main.ts
├── .gitignore
├── env.d.ts // 为环境变量提供类型定义
├── index.html // 入口文件
├── package-lock.json
├── package.json
├── README.md
├── tsconfig.app.json
├── tsconfig.json
├── tsconfig.node.json
└── vite.config.ts
```
