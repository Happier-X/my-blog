## 安装 Nest CLI

Nest CLI 是官方提供的脚手架工具，用于帮助我们创建、开发和维护 Nest 应用程序

建议全局安装：

```bash
npm install -g @nestjs/cli
```

更新（有新版本时可以执行以下命令）：

```bash
npm update -g @nestjs/cli
```

## 创建 Nest 项目

```bash
nest new 你的项目名
```

## 运行项目

```bash
cd 你的项目文件夹
```

```bash
npm run start:dev
```

在浏览器中，打开 `localhost:3000` 查看运行情况。
