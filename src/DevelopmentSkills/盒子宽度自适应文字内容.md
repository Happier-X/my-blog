# 盒子宽度自适应文字内容

## 效果描述

当文字多少不确定时，让盒子宽度自适应文字内容。

## 实现方法

为盒子设置 `width:fit-content;`。

## 效果演示

::: normal-demo 效果演示

```html
<div class="fit-content">我比较长</div>
<div class="fit-content">他好短</div>
```

```css
.fit-content {
  width: fit-content;
  border: 1px solid black;
  margin: 10px;
}
```

:::
