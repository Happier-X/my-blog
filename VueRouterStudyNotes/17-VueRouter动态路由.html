Vue Router еҠЁжҖҒи·Ҝз”ұ

Happier2024е№ҙ10жңҲ29ж—Ҙ...еӨ§зәҰ 2 еҲҶй’ҹејҖеҸ‘VueVue Router

Vue Router еҠЁжҖҒи·Ҝз”ұ

еҜ№и·Ҝз”ұзҡ„ж·»еҠ йҖҡеёёжҳҜйҖҡиҝҮ routes йҖүйЎ№жқҘе®ҢжҲҗзҡ„пјҢдҪҶжҳҜеңЁжҹҗдәӣжғ…еҶөдёӢпјҢдҪ еҸҜиғҪжғіеңЁеә”з”ЁзЁӢеәҸе·Із»ҸиҝҗиЎҢзҡ„ж—¶еҖҷж·»еҠ жҲ–еҲ йҷӨи·Ҝз”ұгҖӮ

еҠЁжҖҒи·Ҝз”ұж·»еҠ 

дҪ еҸҜд»ҘдҪҝз”Ё router.addRoute() ж–№жі•еҠЁжҖҒж·»еҠ и·Ҝз”ұгҖӮ

е®ғеҸӘжіЁеҶҢдёҖдёӘж–°зҡ„и·Ҝз”ұпјҢд№ҹе°ұжҳҜиҜҙпјҢеҰӮжһңж–°еўһеҠ зҡ„и·Ҝз”ұдёҺеҪ“еүҚдҪҚзҪ®зӣёеҢ№й…ҚпјҢе°ұйңҖиҰҒз”Ё router.push() жҲ– router.replace() жқҘжүӢеҠЁеҜјиҲӘпјҢжүҚиғҪжҳҫзӨәиҜҘж–°и·Ҝз”ұгҖӮиҖҢеңЁеҜјиҲӘе®ҲеҚ«дёӯпјҢеҲҷеҸҜд»ҘйҖҡиҝҮиҝ”еӣһж–°зҡ„дҪҚзҪ®жқҘи§ҰеҸ‘йҮҚе®ҡеҗ‘гҖӮ

router.addRoute({ path: '/about', component: About })

addRoute() ж–№жі•иҝҳеҸҜд»ҘжҺҘеҸ—дёҖдёӘеөҢеҘ—зҡ„и·Ҝз”ұеҜ№иұЎпјҢйҖҡиҝҮз»ҷз¬¬дёҖдёӘеҸӮж•°дј йҖ’ name жқҘе®һзҺ°гҖӮ

router.addRoute({ name: 'admin', path: '/admin', component: Admin })
router.addRoute('admin', { path: 'settings', component: AdminSettings })

еҠЁжҖҒи·Ҝз”ұеҲ йҷӨ

йҖҡиҝҮж·»еҠ дёҖдёӘеҗҚз§°еҶІзӘҒзҡ„и·Ҝз”ұгҖӮеҰӮжһңж·»еҠ дёҺзҺ°жңүйҖ”еҫ„еҗҚз§°зӣёеҗҢзҡ„йҖ”еҫ„пјҢдјҡе…ҲеҲ йҷӨи·Ҝз”ұпјҢеҶҚж·»еҠ и·Ҝз”ұгҖӮ

router.addRoute({ path: '/about', name: 'about', component: About })
// иҝҷе°ҶдјҡеҲ йҷӨд№ӢеүҚе·Із»Ҹж·»еҠ зҡ„и·Ҝз”ұпјҢеӣ дёәд»–д»¬е…·жңүзӣёеҗҢзҡ„еҗҚеӯ—дё”еҗҚеӯ—еҝ…йЎ»жҳҜе”ҜдёҖзҡ„
router.addRoute({ path: '/other', name: 'about', component: Other })

еҪ“и·Ҝз”ұжІЎжңүеҗҚз§°ж—¶пјҢеҸҜд»ҘдҪҝз”Ё addRoute() ж–№жі•зҡ„иҝ”еӣһеҖјжқҘеҲ йҷӨи·Ҝз”ұгҖӮ

const removeRoute = router.addRoute(routeRecord)
removeRoute() // еҲ йҷӨи·Ҝз”ұеҰӮжһңеӯҳеңЁзҡ„иҜқ

д№ҹеҸҜд»ҘдҪҝз”Ё router.removeRoute() ж–№жі•ж №жҚ®еҗҚз§°жқҘеҠЁжҖҒеҲ йҷӨи·Ҝз”ұгҖӮ

router.addRoute({ path: '/about', name: 'about', component: About })
// еҲ йҷӨи·Ҝз”ұ
router.removeRoute('about')

жіЁж„ҸпјҡеҪ“и·Ҝз”ұиў«еҲ йҷӨж—¶пјҢжүҖжңүзҡ„еҲ«еҗҚе’Ңеӯҗи·Ҝз”ұд№ҹдјҡиў«еҗҢж—¶еҲ йҷӨгҖӮ

жҹҘзңӢи·Ҝз”ұ

router.hasRoute(и·Ҝз”ұеҗҚз§°)пјҡжЈҖжҹҘи·Ҝз”ұжҳҜеҗҰеӯҳеңЁгҖӮ

router.getRoutes()пјҡиҺ·еҸ–дёҖдёӘеҢ…еҗ«жүҖжңүи·Ҝз”ұи®°еҪ•зҡ„ж•°з»„гҖӮ
