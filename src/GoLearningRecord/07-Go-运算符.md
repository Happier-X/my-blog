运算符用于在程序运行时执行数学或逻辑运算。

Go 内置的运算符有：
+ 算术运算符
+ 关系运算符
+ 逻辑运算符
+ 位运算符
+ 赋值运算符
+ 其他运算符

## 算数运算符

假设 A 为 10，B 为 20

|运算符|描述|实例|
|---|---|---|
|+|相加|A + B 输出结果为 30|
|-|相减|A - B 输出结果为 -10|
|*|相乘|A * B 输出结果为 200|
|/|相除|B / A 输出结果为 2|
|%|求余|B % A 输出结果为 0|
|++|自增|A++ 输出结果为 11|
|--|自减|A-- 输出结果为 9|
