## 使用 Vite 创建

命令：

```shell
npm create vue@latest
```

## 项目结构

```
├── node_modules
├── public
│   └── index.html
├── src
│   ├── assets
│   ├── components
│   ├── App.vue
│   ├── main.js
│   └── shims-vue.d.ts
├── .gitignore
├── index.html
├── package.json
├── README.md
└── tsconfig.json
```

## 项目运行

```shell
npm run dev
```

## 项目打包

```shell
npm run build
```
