http-server

http-server 是一个简单、零配置的命令行 HTTP 服务器，用于快速启动一个静态服务器。

安装

确保已经安装了 Node 和 npm。

npm install -g http-server

使用

在项目目录下运行以下命令。

http-server

常用在于开发环境，用于快速启动一个静态服务器，方便调试和预览，比如我们打包了一个 Vue 项目，在 dist 文件夹下可以启动一个静态服务器，方便预览打包后的效果。
