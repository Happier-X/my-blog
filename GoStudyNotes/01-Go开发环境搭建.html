Go 开发环境搭建

Happier2024年7月9日小于 1 分钟开发Go
此页内容
  • 下载并安装 Go 语言环境
  • 验证安装
  • 配置 GOPROXY
  • 安装代码编辑器

下载并安装 Go 语言环境

访问 Go 语言的官方网站，选择适合操作系统的安装包进行下载并按提示安装

验证安装

在终端中输入 go version 来验证 Go 是否安装成功

配置 GOPROXY

由于国内访问不到默认的 GOPROXY 配置链接，我们需要配置成可访问的

go env -w GOPROXY=https://goproxy.cn,direct

安装代码编辑器

使用 VSCode 安装 Go 扩展即可
